# 🚀 FastAPI React Task Manager

A full-stack task manager web application built using FastAPI, React, and TypeScript.

This project is part of my learning journey to become a better backend developer, focusing on building APIs, handling data, and integrating a modern frontend.

---

## 🧠 Goals of this project

- Learn FastAPI from scratch
- Understand how frontend and backend communicate
- Practice building RESTful APIs
- Work with real-world project structure
- Prepare for full-stack development

---

## 🛠️ Tech Stack

### Backend
- FastAPI
- Python

### Frontend
- React
- TypeScript
